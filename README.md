# Low Level Design (LLD) in Java

This repository contains implementations of common Low Level Design (LLD) patterns using Java.

## Patterns Covered
- Command Pattern (Bad vs Good Design)
- Iterator Pattern (Bad vs Good Design)
- Observer Pattern
- Strategy Pattern
- Memento Pattern

## Goal
To understand design patterns deeply by implementing them with simple and clear examples.

## Note
This repository focuses on learning and understanding core concepts before applying them in real-world systems.
